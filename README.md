# TimeSync

[![Live Demo](https://img.shields.io/badge/Live_Demo-timesync--tool.surge.sh-brightgreen?style=for-the-badge)](https://timesync-tool.surge.sh)

> Find the best meeting time across any time zones. Visual, instant, shareable.

**Try it now:** [timesync-tool.surge.sh](https://timesync-tool.surge.sh)

## Features

- **Visual 24-hour timeline** for each timezone — see work hours vs night at a glance
- **Drag slider** to pick a time, see everyone's local time instantly
- **Quick-add buttons** for NYC, London, Beijing, Tokyo, Sydney, Berlin, Mumbai
- **Shareable link** — generates URL with all timezones and selected hour
- **Copy meeting card** — one-click copy of meeting times for all participants
- **URL parameters** — `?zones=America/New_York,Asia/Tokyo&hour=14`

## Why?

Every remote team wastes time in Slack asking "what time is 3pm your time?" World Time Buddy is cluttered. Time.zone is simple, visual, and generates shareable links.

## Tech

Single HTML file. Zero dependencies. Pure Intl.DateTimeFormat API.
